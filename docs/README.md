# AI Office Agent — 项目文档目录

本目录存放项目的长期记忆文档，**不依赖聊天记录**。

## 文档清单

| 文档 | 作用 |
| --- | --- |
| `ProjectMemory.md` | 项目长期记忆与整体状态（开发前必读） |
| `CHANGELOG.md` | 变更日志，按版本倒序 |
| `Roadmap.md` | 路线图与开发规划 |
| `DevelopmentRules.md` | 开发规则（长期纪律） |

## 使用规范

详见 `DevelopmentRules.md`。要点：

1. 每次开发前先读这四个文档。
2. 每完成一个任务即停止开发。
3. 停止前必须更新 `ProjectMemory.md` 与 `CHANGELOG.md`。
4. 需求变化时同步更新 `ProjectMemory.md`。
